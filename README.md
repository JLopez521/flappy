# INSTRUCTIONS

When you download our game into your files there are a few things you must do!

1. Download python version 3.13.1 - make sure you download the windows terminal version and not the actual app! We'll link it below as well. (This part takes a while so give it some time)

https://www.python.org/downloads/release/python-3131/

2. Once python is downloaded download pygame on your terminal. Simply type " pip install pygame " and it should automatically download the latest version of pygame.

3. Once you have all of these downloaded it's time to head to the terminal and start up the game. Normally you would just do python main.py but this one is a bit different. Instead go into the folder before the "flappy" folder (this changes based on where you downloaded the game), and type in python " .\flappy\main.py ".

4. That's it! Enjoy the game get as far as you can and get a new high score!
